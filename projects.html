Development Projects

CryptoPricer

The CryptoPricer web app is a tool to convert the value one of many popular cryptocurrencies into a realtime traditional currency equivalent. The tool also displays a graph tracking the value of that cryptocurrency over the last 100 days.

Chatter

Chatter is an online chat room where users can send and receive messages across devices, and includes customizable screen names to differentiate users.

Lyric Finder

Lyric Finder takes .mp3 and .mp4s and uses the AudD music recognition API to ID music found in the submitted file. In addition to identifying songs the API also provides details about the song including the artist, its lyrics, and more.
